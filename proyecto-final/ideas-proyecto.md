# Ideas para el Proyecto Final

## Presentación

Este archivo contiene ideas de proyectos finales para el curso de Algoritmia y Programación.

Cada idea puede adaptarse según el nivel del estudiante y el tiempo disponible.

# Recomendación para elegir proyecto

Elige un proyecto que puedas explicar con claridad.

Un buen proyecto no es el más grande, sino el que está bien organizado, funciona correctamente y demuestra los temas del curso.

---

# Preguntas para seleccionar una idea

Antes de elegir, responde:

1. ¿Qué problema resuelve?
2. ¿Qué datos necesita?
3. ¿Qué clases puedo crear?
4. ¿Qué métodos necesito?
5. ¿Qué validaciones debe tener?
6. ¿Qué información mostrará al final?
7. ¿Puedo terminarlo en el tiempo disponible?
