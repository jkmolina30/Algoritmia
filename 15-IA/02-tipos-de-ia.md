# 02. Tipos de Inteligencia Artificial

## Presentación

Existen diferentes formas de clasificar la inteligencia artificial. Para este curso se explicarán de manera sencilla los tipos más importantes que un estudiante puede encontrar.

---

# 1. IA según su capacidad

## 1.1. IA estrecha o débil

Es una IA diseñada para realizar tareas específicas.

Ejemplos:

- Un asistente que responde preguntas.
- Un sistema que recomienda canciones.
- Una herramienta que traduce textos.
- Una IA que reconoce rostros.
- Un asistente que ayuda a escribir código.

La mayoría de las IA actuales son de este tipo.

---

## 1.2. IA general

Sería una IA capaz de razonar y aprender de forma amplia, como una persona, en muchos contextos diferentes.

Actualmente, esta idea sigue siendo un objetivo de investigación y discusión, no una herramienta cotidiana disponible para cualquier tarea humana.

---

# 2. IA según el tipo de tarea

## 2.1. IA de texto

Genera, resume, corrige o explica textos.

Usos:

- Redactar correos.
- Resumir documentos.
- Explicar conceptos.
- Crear guías.
- Hacer preguntas de estudio.
- Ayudar con documentación.

---

## 2.2. IA de código

Ayuda a programar.

Usos:

- Explicar errores.
- Generar ejemplos.
- Sugerir mejoras.
- Documentar código.
- Crear pruebas.
- Explicar algoritmos.

Advertencia:

```text
Una IA puede generar código incorrecto. Siempre se debe probar.
```

---

## 2.3. IA de imágenes

Genera o edita imágenes.

Usos:

- Crear ilustraciones.
- Diseñar portadas.
- Generar diagramas visuales.
- Apoyar presentaciones.
- Crear material didáctico.

---

## 2.4. IA de audio

Trabaja con sonido o voz.

Usos:

- Transcribir audio.
- Generar voz.
- Traducir conversaciones.
- Crear narraciones.

---

## 2.5. IA de video

Ayuda a crear o editar video.

Usos:

- Generar clips.
- Editar contenido.
- Crear explicaciones visuales.
- Generar subtítulos.

---

## 2.6. IA multimodal

Una IA multimodal puede trabajar con varios tipos de entrada o salida.

Por ejemplo:

- Texto.
- Imagen.
- Audio.
- Video.
- Archivos.
- Código.

Ejemplo de uso:

```text
Subir una imagen de un error en Java y pedir explicación.
```

---

# 3. IA según su uso educativo

## 3.1. IA como tutor

Ayuda a explicar temas.

Ejemplo:

```text
Explícame qué es un arreglo en Java como si estuviera iniciando.
```

---

## 3.2. IA como evaluador

Puede revisar respuestas, código o explicaciones.

Ejemplo:

```text
Revisa mi explicación sobre ciclos while y dime qué está incompleto.
```

---

## 3.3. IA como generador de práctica

Puede crear ejercicios.

Ejemplo:

```text
Créame 10 ejercicios de condicionales en Java, de menor a mayor dificultad.
```

---

## 3.4. IA como depurador

Puede ayudar a encontrar errores.

Ejemplo:

```text
Este código no compila. Explícame el error línea por línea.
```

---

## 3.5. IA como asistente de documentación

Puede ayudar a escribir README, comentarios y guías.

Ejemplo:

```text
Ayúdame a escribir un README para mi proyecto de notas en Java.
```

---

# 4. Riesgos según el tipo de IA

| Tipo de IA | Riesgo principal |
|---|---|
| Texto | Inventar información |
| Código | Generar código incorrecto o no explicado |
| Imagen | Crear contenido engañoso |
| Audio | Imitación de voces |
| Video | Manipulación visual |
| Multimodal | Mezclar errores de varias fuentes |

---

# 5. Recomendación para estudiantes

Antes de usar una herramienta de IA, pregúntate:

```text
¿Para qué la necesito?
¿Me ayuda a aprender?
¿Voy a verificar la respuesta?
¿Puedo explicar el resultado?
¿Estoy respetando las reglas del curso?
```

---

## Conclusión

No todas las herramientas de IA hacen lo mismo. Algunas sirven para escribir, otras para programar, otras para diseñar y otras para analizar información. Lo importante es elegir la herramienta adecuada y usarla con responsabilidad.
