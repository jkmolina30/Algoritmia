# Reto integrador

## Sistema básico de notas con arreglos

Crear un programa que permita registrar la información de varios estudiantes.

El programa debe:

1. Preguntar cuántos estudiantes se van a registrar.
2. Crear dos arreglos:
   - `String[] nombres`
   - `double[] notas`
3. Leer y validar cada nota.
4. Calcular el promedio.
5. Mostrar:
   - Todos los estudiantes.
   - Promedio del grupo.
   - Cantidad de aprobados.
   - Cantidad de reprobados.
   - Nota mayor.
   - Nota menor.

| Criterio | Descripción |

| Arreglos | Declara y usa arreglos correctamente |
| Ciclos | Recorre arreglos con `for` |
| Validaciones | Valida notas |
| Condicionales | Cuenta aprobados y reprobados |
| Acumuladores | Calcula suma y promedio |
| Claridad | Usa nombres adecuados |
| Orden | Código indentado |
