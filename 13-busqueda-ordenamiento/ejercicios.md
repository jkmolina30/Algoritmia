# Reto integrador

## Sistema de notas con búsqueda y ordenamiento

Crear un programa en Java que permita:

1. Registrar nombres y notas de estudiantes.
2. Buscar un estudiante por nombre.
3. Mostrar la nota del estudiante encontrado.
4. Ordenar las notas de menor a mayor.
5. Mostrar el promedio del grupo.
6. Mostrar la nota mayor y la nota menor.

Requisitos:

- Usar arreglos o ArrayList.
- Usar búsqueda secuencial para nombres.
- Usar ordenamiento burbuja o selección para notas.
- Validar que las notas estén entre 0.0 y 5.0.
