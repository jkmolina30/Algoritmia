# Algoritmia para Ingeniería de Sistemas

Curso académico diseñado para estudiantes de primer semestre del Programa de Ingeniería de Sistemas.

## Descripción

La algoritmia constituye el fundamento del pensamiento computacional y de la programación. Este curso ha sido diseñado para desarrollar en los estudiantes la capacidad de analizar problemas, diseñar soluciones mediante algoritmos y posteriormente implementarlas utilizando el lenguaje de programación Java.

A lo largo del curso se abordarán conceptos fundamentales como el análisis de problemas, el uso de pseudocódigo, diagramas de flujo, estructuras de control, funciones, arreglos, matrices y algoritmos clásicos, permitiendo una transición progresiva desde la lógica computacional hasta la programación orientada a objetos.

El enfoque principal es aprender a resolver problemas antes de programarlos, fortaleciendo las habilidades de razonamiento lógico y pensamiento estructurado que todo ingeniero de sistemas debe desarrollar.

---

## Objetivos del Curso

Al finalizar el curso, el estudiante estará en capacidad de:

* Comprender los fundamentos de la algoritmia y el pensamiento computacional.
* Analizar problemas y modelar soluciones mediante algoritmos.
* Diseñar soluciones utilizando pseudocódigo y diagramas de flujo.
* Aplicar estructuras de control para la resolución de problemas.
* Utilizar funciones y procedimientos para modularizar soluciones.
* Implementar algoritmos utilizando el lenguaje de programación Java.
* Desarrollar programas básicos aplicando buenas prácticas de programación.

---

## Metodología

Cada unidad temática está estructurada bajo la siguiente metodología:

1. Introducción conceptual.
2. Explicación teórica.
3. Ejemplos guiados paso a paso.
4. Desarrollo en pseudocódigo.
5. Pruebas de escritorio.
6. Implementación en Java.
7. Ejercicios de práctica.
8. Retos de aplicación.
9. Talleres evaluativos.

El propósito es que el estudiante comprenda primero la lógica de solución antes de enfrentarse a la sintaxis de un lenguaje de programación.

## Contenido General

### Unidad 1. Introducción a la Algoritmia

* Concepto de algoritmo.
* Análisis de problemas.
* Entradas, procesos y salidas.

### Unidad 2. Pseudocódigo y Diagramas de Flujo

* Variables y constantes.
* Tipos de datos.
* Operadores.
* Diseño de algoritmos.

### Unidad 3. Estructuras Condicionales

* Condicional simple.
* Condicional doble.
* Condicional múltiple.

### Unidad 4. Estructuras Repetitivas

* Ciclo Mientras.
* Ciclo Para.
* Ciclo Repetir-Hasta.

### Unidad 5. Funciones y Procedimientos

* Modularización.
* Parámetros.
* Valores de retorno.

### Unidad 6. Arreglos

* Declaración.
* Recorrido.
* Búsqueda.

### Unidad 7. Matrices

* Estructuras bidimensionales.
* Operaciones básicas.

### Unidad 8. Algoritmos Fundamentales

* Búsqueda secuencial.
* Búsqueda binaria.
* Ordenamiento.

### Unidad 9. Introducción a Java

* Sintaxis básica.
* Entrada y salida de datos.
* Estructuras de control.

### Unidad 10. Programación Orientada a Objetos

* Clases.
* Objetos.
* Métodos.
* Encapsulamiento.
