# Hardware y Software

## 1. Introducción

Todo computador está formado por dos grandes componentes:

```text
Hardware + Software
```

El hardware corresponde a la parte física.

El software corresponde a los programas e instrucciones.

Ambos son necesarios para que un computador funcione.

---

## 2. ¿Qué es hardware?

El **hardware** es todo aquello que se puede tocar físicamente en un computador o dispositivo tecnológico.

Ejemplos:

* Teclado.
* Mouse.
* Pantalla.
* CPU.
* Memoria RAM.
* Disco duro.
* Cámara.
* Parlantes.
* Impresora.
* Placa base.
* Tarjeta gráfica.

En palabras sencillas, el hardware es el cuerpo físico del computador.

---

## 3. Clasificación del hardware

El hardware puede clasificarse según su función.

### 3.1. Hardware de entrada

Permite ingresar datos al computador.

Ejemplos:

* Teclado.
* Mouse.
* Cámara.
* Micrófono.
* Escáner.
* Pantalla táctil.

### 3.2. Hardware de salida

Permite mostrar resultados.

Ejemplos:

* Monitor.
* Impresora.
* Parlantes.
* Proyector.
* Audífonos.

### 3.3. Hardware de procesamiento

Permite realizar operaciones y ejecutar instrucciones.

Ejemplos:

* CPU.
* GPU.
* Placa base.

### 3.4. Hardware de almacenamiento

Permite guardar datos.

Ejemplos:

* Disco duro.
* Unidad de estado sólido.
* Memoria USB.
* Tarjeta SD.

### 3.5. Hardware de comunicación

Permite conectar el computador con otros dispositivos o redes.

Ejemplos:

* Tarjeta de red.
* Adaptador Wi-Fi.
* Bluetooth.
* Módem.
* Router.

---

## 4. ¿Qué es software?

El **software** es el conjunto de programas, instrucciones y datos que permiten que el hardware realice tareas.

El software no se puede tocar físicamente.

Ejemplos:

* Windows.
* Linux.
* Android.
* Microsoft Word.
* Google Chrome.
* NetBeans.
* Java.
* PSeInt.
* Aplicaciones móviles.
* Videojuegos.

En palabras sencillas, el software es la parte lógica del computador.

---

## 5. Clasificación del software

El software puede clasificarse en tres grandes grupos.

---

## 5.1. Software de sistema

Es el software que permite que el computador funcione y administre sus recursos.

Ejemplos:

* Windows.
* Linux.
* macOS.
* Android.
* iOS.

El software de sistema controla elementos como:

* Memoria.
* Archivos.
* Procesos.
* Usuarios.
* Dispositivos.
* Programas instalados.

---

## 5.2. Software de aplicación

Es el software que permite al usuario realizar tareas específicas.

Ejemplos:

| Aplicación | Uso                           |
| ---------- | ----------------------------- |
| Word       | Escribir documentos           |
| Excel      | Trabajar con hojas de cálculo |
| Chrome     | Navegar en internet           |
| WhatsApp   | Comunicarse                   |
| Spotify    | Escuchar música               |
| Canva      | Diseñar contenido             |
| Moodle     | Gestionar cursos              |

---

## 5.3. Software de desarrollo

Es el software utilizado para crear otros programas.

Ejemplos:

* NetBeans.
* Eclipse.
* Visual Studio Code.
* IntelliJ IDEA.
* PSeInt.
* Git.
* JDK.
* Compiladores.
* Editores de código.

En este curso se usará software de desarrollo para crear programas en Java.

---

## 6. Relación entre hardware y software

El hardware y el software trabajan juntos.

Un computador con buen hardware pero sin software no puede realizar tareas útiles.

Un software sin hardware no puede ejecutarse.

### Ejemplo

Para escribir un programa en Java se necesita:

| Elemento                               | Tipo     |
| -------------------------------------- | -------- |
| Computador físico                      | Hardware |
| Teclado para escribir código           | Hardware |
| Monitor para ver el código             | Hardware |
| Sistema operativo                      | Software |
| JDK                                    | Software |
| NetBeans                               | Software |
| Programa Java creado por el estudiante | Software |

---

## 7. Ejemplo práctico

Supongamos que un estudiante escribe un programa para calcular el promedio de notas.

### Hardware involucrado

* Teclado: permite ingresar las notas.
* CPU: procesa las operaciones.
* RAM: guarda temporalmente los valores.
* Pantalla: muestra el resultado.
* Disco: guarda el archivo del programa.

### Software involucrado

* Sistema operativo.
* NetBeans.
* JDK.
* Programa Java.
* Librerías de Java.

---

## 8. ¿Dónde se ubica Java?

Java es parte del software de desarrollo.

Para trabajar con Java se necesita instalar el JDK, que permite compilar y ejecutar programas.

También se puede usar un IDE como NetBeans, Eclipse o Visual Studio Code para escribir el código de manera organizada.

---

## 9. ¿Dónde se ubica PSeInt?

PSeInt también es software de desarrollo y aprendizaje.

Permite escribir pseudocódigo y practicar lógica de programación.

Se usa principalmente al inicio del curso para comprender algoritmos antes de pasar a Java.

---

## 10. Diferencias entre hardware y software

| Aspecto        | Hardware                    | Software                          |
| -------------- | --------------------------- | --------------------------------- |
| Naturaleza     | Físico                      | Lógico                            |
| Se puede tocar | Sí                          | No                                |
| Ejemplos       | Teclado, pantalla, CPU      | Windows, Java, NetBeans           |
| Función        | Ejecutar físicamente tareas | Dar instrucciones                 |
| Daño común     | Falla física                | Error, virus, mala configuración  |
| Actualización  | Cambio de piezas            | Instalación o mejora de programas |

---

## 11. Importancia para un programador

Un programador debe entender la diferencia entre hardware y software porque sus programas se ejecutan sobre una máquina real.

Por ejemplo:

* Si un programa consume mucha memoria, puede afectar el rendimiento.
* Si un archivo no se guarda correctamente, puede perderse información.
* Si un programa necesita internet, depende del hardware y software de red.
* Si un programa tiene errores, puede fallar aunque el hardware esté bien.

---

## 12. Resumen

El hardware es la parte física del computador. El software es la parte lógica formada por programas e instrucciones. Ambos trabajan juntos para que un sistema computacional funcione. En programación, el estudiante crea software que será ejecutado por el hardware a través del sistema operativo y las herramientas de desarrollo.
