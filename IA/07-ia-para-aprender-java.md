# 07. IA para Aprender Java

## Presentación

La IA puede ser una excelente herramienta para aprender Java si se usa correctamente.

En lugar de pedir respuestas completas, el estudiante puede usarla como tutor, revisor, generador de práctica y acompañante de estudio.

---

## 1. Usar IA para entender conceptos

Prompt recomendado:

```text
Explícame qué es una variable en Java como si fuera principiante. Usa un ejemplo con edad y otro con nota final.
```

---

## 2. Usar IA para comparar conceptos

```text
Compara arreglo y ArrayList en Java. Usa una tabla y un ejemplo corto de cada uno.
```

---

## 3. Usar IA para practicar

```text
Créame 10 ejercicios de condicionales en Java para principiantes. No incluyas las soluciones.
```

---

## 4. Usar IA para pedir pistas

```text
Tengo este ejercicio: leer 5 notas y calcular el promedio.
No me des el código completo. Dame una pista para empezar.
```

---

## 5. Usar IA para revisar código

```text
Este código Java no funciona. Explícame el error y dime cómo corregirlo sin cambiar toda mi lógica.

[pegar código]
```

---

## 6. Usar IA para hacer prueba de escritorio

```text
Haz una prueba de escritorio para este código Java. Muestra cómo cambian las variables en cada iteración.

[pegar código]
```

---

## 7. Usar IA para prepararse para un parcial

```text
Hazme 15 preguntas de práctica sobre ciclos en Java. Incluye preguntas conceptuales y ejercicios cortos.
```

---

## 8. Usar IA para explicar errores comunes

```text
Explícame por qué aparece ArrayIndexOutOfBoundsException en Java y dame un ejemplo sencillo.
```

---

## 9. Usar IA para mejorar documentación

```text
Ayúdame a mejorar el README de mi proyecto Java. No inventes funciones que no existen.

[pegar README]
```

---

## 10. Usar IA para sustentar un proyecto

```text
Ayúdame a preparar una explicación oral de mi proyecto Java. El proyecto tiene clases Estudiante, Curso y Principal.
```

---

# 11. Temas de Java donde la IA puede ayudar

| Tema | Cómo puede ayudar |
|---|---|
| Variables | Explicar tipos de datos |
| Condicionales | Crear casos de práctica |
| Ciclos | Hacer pruebas de escritorio |
| Arreglos | Explicar índices |
| Matrices | Mostrar recorrido con ciclos anidados |
| Métodos | Explicar parámetros y retorno |
| POO | Explicar clases y objetos |
| GitHub | Ayudar con README y comandos |

---

# 12. Lo que el estudiante debe evitar

Evitar prompts como:

```text
Hazme el código completo.
Dame el proyecto final.
Resuelve todo sin explicar.
Haz el código para que parezca mío.
```

---

# 13. Método de estudio con IA

Usa esta ruta:

```text
1. Intento resolver solo.
2. Si me bloqueo, pido una pista.
3. Escribo mi solución.
4. Pido revisión.
5. Corrijo.
6. Pruebo.
7. Explico con mis palabras.
```

---

# 14. Ejemplo completo de apoyo

Prompt:

```text
Estoy aprendiendo arreglos en Java. Tengo que leer 5 notas y calcular el promedio. Ya intenté hacerlo con un for, pero no entiendo cómo acumular la suma. Explícame solo la parte del acumulador con un ejemplo pequeño.
```

Este prompt es adecuado porque:

- Dice el tema.
- Dice el problema.
- Explica qué parte no entiende.
- No pide la tarea completa.
- Busca aprender.

---

## Conclusión

La IA puede convertirse en un tutor útil si el estudiante la usa para pensar mejor, practicar más y corregir errores, no para evitar el aprendizaje.
