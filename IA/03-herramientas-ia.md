# 03. Herramientas de IA y para qué sirven

## Presentación

Existen muchas herramientas de inteligencia artificial. Algunas sirven para conversar, otras para programar, crear imágenes, resumir, diseñar, traducir o buscar información.

Esta lista no es un ranking definitivo. Las herramientas cambian con el tiempo, aparecen nuevas opciones y algunas funciones pueden variar según el país, el plan o la versión.

---

# 1. ChatGPT

## Uso principal

Asistente conversacional de IA.

Puede ayudar a:

- Explicar conceptos.
- Resolver dudas.
- Crear ejemplos.
- Revisar textos.
- Apoyar programación.
- Generar ideas.
- Ayudar con documentación.

## Uso educativo recomendado

```text
Explícame los ciclos for en Java con un ejemplo sencillo.
```

---

# 2. Google Gemini

## Uso principal

Asistente de IA integrado al ecosistema de Google.

Puede ayudar a:

- Responder preguntas.
- Trabajar con texto.
- Apoyar búsquedas.
- Resumir información.
- Integrarse con herramientas de productividad, según disponibilidad.

NOTA: Cuando se tienen cuentas asociadas a Google, como gmail, podemos hacer uso de esta herramienta en su version gratuita

---

# 3. Microsoft Copilot

## Uso principal

Asistente de IA de Microsoft.

Puede ayudar a:

- Buscar información.
- Redactar contenido.
- Apoyar tareas de productividad.
- Trabajar en algunos entornos de Microsoft.
- Generar ideas y explicaciones.

NOTA: Cuando se tienen cuentas asociadas a microsoft, como las que son de uso institucional podemos hacer uso de esta herramienta en su version gratuita 

---

# 4. Claude

## Uso principal

Asistente conversacional útil para lectura, análisis y escritura.

Puede ayudar a:

- Analizar documentos.
- Explicar conceptos.
- Redactar textos.
- Revisar ideas.
- Apoyar tareas de programación.

---

# 5. GitHub Copilot

## Uso principal

Asistente para programadores.

Puede ayudar a:

- Sugerir código.
- Autocompletar funciones.
- Proponer soluciones.
- Explicar fragmentos.
- Generar pruebas.

## Advertencia

No se debe aceptar código sin revisarlo.  
El estudiante debe entender, probar y adaptar lo que sugiera la herramienta.

---

# 6. Perplexity

## Uso principal

Búsqueda asistida con IA.

Puede ayudar a:

- Buscar información.
- Resumir fuentes.
- Comparar conceptos.
- Encontrar referencias.

## Recomendación

Verificar las fuentes citadas y no quedarse solo con el resumen.

---

# 7. Canva con IA

## Uso principal

Diseño y creación visual.

Puede ayudar a:

- Crear presentaciones.
- Diseñar afiches.
- Crear imágenes.
- Organizar contenido visual.
- Preparar material educativo.

---

# 8. Grammarly o herramientas similares

## Uso principal

Corrección y mejora de escritura.

Puede ayudar a:

- Corregir redacción.
- Mejorar claridad.
- Revisar gramática.
- Ajustar tono.

---

# 9. DeepL y traductores con IA

## Uso principal

Traducción y mejora de textos en otros idiomas.

Puede ayudar a:

- Traducir documentos.
- Mejorar redacción en inglés.
- Comparar versiones de textos.

---

# 10. Notion AI u otras IA de productividad

## Uso principal

Organización de notas, resúmenes y planificación.

Puede ayudar a:

- Resumir apuntes.
- Crear listas.
- Organizar ideas.
- Planear proyectos.

---

# 11. Herramientas de generación de imágenes

Ejemplos de uso:

- Crear ilustraciones.
- Diseñar recursos visuales.
- Generar ideas para presentaciones.
- Apoyar actividades creativas.

Advertencia:

```text
No usar imágenes generadas para engañar, suplantar personas o crear contenido ofensivo.
```

---

# 13. ¿Cuál herramienta usar?

Depende de la tarea.

| Necesidad | Herramienta sugerida |
|---|---|
| Entender un tema | ChatGPT, Gemini, Claude |
| Buscar con fuentes | Perplexity, buscadores académicos |
| Programar | ChatGPT, GitHub Copilot, Claude |
| Diseñar una presentación | Canva |
| Corregir redacción | Grammarly |
| Traducir | DeepL |
| Organizar apuntes | Notion AI |

---

# 14. Recomendación final

No existe una herramienta perfecta para todo.  
Lo importante es saber preguntar, verificar, comparar y adaptar la respuesta al contexto académico.
