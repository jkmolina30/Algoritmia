# 10. Taller Práctico: Uso Responsable de IA en Programación

## Objetivo

Aprender a usar la inteligencia artificial como apoyo para estudiar programación, sin reemplazar el razonamiento propio.

---

## Competencias a fortalecer

Al finalizar el taller, el estudiante podrá:

- Formular mejores prompts.
- Usar IA para comprender conceptos.
- Pedir pistas sin copiar soluciones completas.
- Revisar código con ayuda de IA.
- Verificar respuestas.
- Declarar el uso de IA de forma honesta.

---

# Parte 1. Preguntas iniciales

Responder:

1. ¿Qué herramientas de IA conoces?
2. ¿Para qué las has usado?
3. ¿Qué riesgos tiene usar IA en tareas académicas?
4. ¿Cuál es la diferencia entre usar IA para aprender y usar IA para copiar?
5. ¿Por qué es importante poder explicar el código?

---

# Parte 2. Mejorar prompts

## Actividad 1

Transformar este prompt débil:

```text
Explícame Java.
```

En un prompt claro para aprender variables.

---

## Actividad 2

Transformar este prompt débil:

```text
Hazme un programa de notas.
```

En un prompt que pida orientación paso a paso sin entregar toda la solución.

---

## Actividad 3

Transformar este prompt débil:

```text
Arregla mi código.
```

En un prompt que solicite revisión, explicación del error y corrección guiada.

---

# Parte 3. Uso de IA para entender código

Usar este código:

```java
public class SumaNumeros {
    public static void main(String[] args) {
        int suma = 0;

        for (int i = 1; i <= 5; i++) {
            suma = suma + i;
        }

        System.out.println("La suma es: " + suma);
    }
}
```

Pedir a la IA:

```text
Explícame este código línea por línea y haz una prueba de escritorio.
```

Luego responder:

1. ¿Qué hace el programa?
2. ¿Cuánto vale `suma` al final?
3. ¿Cuántas veces se ejecuta el ciclo?
4. ¿Qué hace `i++`?
5. ¿Qué aprendiste?

---

# Parte 4. Revisión crítica

Pedir a la IA que resuelva este ejercicio:

```text
Leer tres notas y calcular el promedio.
```

Luego verificar:

- ¿El código compila?
- ¿Usa paréntesis correctamente?
- ¿Valida las notas?
- ¿Usa temas vistos en clase?
- ¿Se entiende la explicación?
- ¿Qué mejorarías?

---

# Parte 5. Prompt responsable

Escribir un prompt para pedir ayuda con este ejercicio, sin solicitar la solución completa:

```text
Crear un programa en Java que lea 5 notas, las guarde en un arreglo y calcule el promedio.
```

Debe incluir:

- Contexto.
- Nivel.
- Tema.
- Restricción.
- Formato esperado.

---

# Parte 6. Declaración de uso de IA

Completar:

```text
Usé IA para:
[explicar uso]

No usé IA para:
[explicar límites]

Verifiqué la respuesta mediante:
[explicar cómo verifiqué]

Puedo explicar:
[explicar qué aprendí]
```

---

# Parte 7. Reflexión final

Responder:

1. ¿Qué aprendiste sobre el uso de IA?
2. ¿Qué harás diferente la próxima vez?
3. ¿Qué prompt te funcionó mejor?
4. ¿Qué error podrías cometer si confías demasiado en la IA?
5. ¿Cómo puedes usar IA sin dejar de aprender?
