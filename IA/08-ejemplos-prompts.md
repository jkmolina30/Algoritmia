# 08. Ejemplos de Prompts para Estudiantes

## Presentación

Este archivo reúne ejemplos de prompts que los estudiantes pueden usar para aprender programación de forma responsable.

---

# 1. Prompt para entender un tema

```text
Actúa como docente de programación.
Estoy aprendiendo condicionales en Java.
Explícame qué es if, else if y else con ejemplos sencillos.
Mi nivel es principiante.
```

---

# 2. Prompt para pedir ejemplos

```text
Dame 3 ejemplos de ciclos for en Java:
1. Mostrar números del 1 al 10.
2. Mostrar una tabla de multiplicar.
3. Calcular el promedio de 5 notas.
Explica cada ejemplo línea por línea.
```

---

# 3. Prompt para pedir ejercicios

```text
Créame 10 ejercicios de arreglos en Java para principiantes.
No incluyas las soluciones.
Ordénalos de menor a mayor dificultad.
```

---

# 4. Prompt para pedir pistas

```text
Tengo este ejercicio:

[pegar enunciado]

No me des la solución completa.
Dame una pista para empezar y ayúdame a identificar entrada, proceso y salida.
```

---

# 5. Prompt para revisar código

```text
Estoy aprendiendo Java.
Este es mi código:

[pegar código]

Necesito que me digas:
1. Si compila.
2. Qué errores tiene.
3. Cómo corregirlos.
4. Qué parte de la lógica debo mejorar.
No me des una solución totalmente nueva si no es necesario.
```

---

# 6. Prompt para explicar errores

```text
Explícame este error de Java como si fuera principiante:

[pegar error]

Dime:
1. Qué significa.
2. Por qué ocurre.
3. Cómo puedo corregirlo.
4. Cómo evitarlo en el futuro.
```

---

# 7. Prompt para prueba de escritorio

```text
Haz una prueba de escritorio del siguiente código Java.
Usa una tabla con las variables principales.

[pegar código]
```

---

# 8. Prompt para estudiar antes del parcial

```text
Estoy preparando un parcial de Java sobre variables, condicionales y ciclos.
Hazme una guía de estudio con:
1. Conceptos clave.
2. Errores comunes.
3. 10 preguntas de práctica.
4. 5 ejercicios de código.
No incluyas respuestas hasta que te las pida.
```

---

# 9. Prompt para explicar POO

```text
Explícame clases y objetos en Java usando un ejemplo de Estudiante.
Incluye clase, atributos, método, objeto y explicación línea por línea.
```

---

# 10. Prompt para mejorar un proyecto

```text
Este es mi proyecto Java:

[describir proyecto]

Quiero mejorarlo.
Dime qué métodos podría crear, qué clases harían falta y qué validaciones debería agregar.
No escribas todo el código completo.
```

---

# 11. Prompt para GitHub

```text
Ayúdame a escribir un README para mi proyecto Java.
El proyecto se llama Sistema de Notas.
Tiene registro de estudiantes, notas, promedio y búsqueda.
Incluye descripción, funcionalidades, tecnologías y cómo ejecutar.
```

---

# 12. Prompt para comparar mi solución

```text
Este es mi código:

[pegar código]

Compáralo con una solución más ordenada.
Dime qué puedo mejorar, pero explica cada mejora.
```

---

# 13. Prompt para evitar copiar

```text
No quiero que me des la solución completa.
Quiero que me hagas preguntas para guiarme hasta resolver este ejercicio por mi cuenta.

[pegar ejercicio]
```

---

# 14. Prompt para pedir retroalimentación

```text
Revisa esta explicación que escribí sobre ciclos while:

[pegar explicación]

Dime:
1. Qué está correcto.
2. Qué falta.
3. Qué ejemplo podría agregar.
4. Cómo mejorar la claridad.
```

---

# 15. Prompt para generar variantes

```text
Dame 5 variantes de este ejercicio para practicar más:

[pegar ejercicio]

Mantén el mismo tema, pero cambia el contexto.
```

---

# 16. Prompt para crear casos de prueba

```text
Tengo un programa que valida notas entre 0.0 y 5.0.
Dame casos de prueba válidos e inválidos para verificar si funciona correctamente.
```

---

# 17. Prompt para preparar sustentación

```text
Tengo que sustentar un proyecto Java sobre inventario.
Ayúdame a preparar un guion de 5 minutos donde explique:
1. Problema.
2. Clases.
3. Métodos.
4. Uso de ArrayList.
5. Validaciones.
6. Qué aprendí.
```

---

## Recomendación final

Un buen prompt debe ayudar al estudiante a aprender.  
Si el prompt solo busca obtener una respuesta para copiar, no está cumpliendo una función formativa.
