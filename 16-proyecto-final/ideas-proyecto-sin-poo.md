# Ideas de Proyecto Final sin POO

## Presentación

Estas ideas están pensadas para trabajar con programación estructurada en Java, sin exigir Programación Orientada a Objetos.

---

## 1. Sistema de notas

Estructuras sugeridas:

```java
ArrayList<String> nombres;
ArrayList<Double> notas;
```

Funcionalidades:

- Registrar estudiante.
- Registrar nota.
- Validar nota.
- Mostrar estudiantes.
- Buscar estudiante.
- Calcular promedio.
- Contar aprobados y reprobados.

---

## 2. Inventario básico

Estructuras sugeridas:

```java
ArrayList<String> productos;
ArrayList<Double> precios;
ArrayList<Integer> cantidades;
```

Funcionalidades:

- Registrar producto.
- Mostrar inventario.
- Buscar producto.
- Calcular valor total.
- Eliminar producto.

---

## 3. Control de gastos

Estructuras sugeridas:

```java
ArrayList<String> categorias;
ArrayList<Double> valores;
```

Funcionalidades:

- Registrar gasto.
- Mostrar gastos.
- Calcular total.
- Calcular promedio.
- Buscar por categoría.
- Mostrar gasto mayor.

---

## 4. Agenda de tareas

Estructuras sugeridas:

```java
ArrayList<String> tareas;
ArrayList<Boolean> realizadas;
```

Funcionalidades:

- Agregar tarea.
- Mostrar tareas.
- Buscar tarea.
- Marcar como realizada.
- Eliminar tarea.
- Contar pendientes.

---

## 5. Sistema de ventas básico

Estructuras sugeridas:

```java
ArrayList<String> productos;
ArrayList<Integer> cantidades;
ArrayList<Double> precios;
```

Funcionalidades:

- Registrar venta.
- Calcular subtotal.
- Calcular IVA.
- Mostrar total.
- Buscar producto vendido.
