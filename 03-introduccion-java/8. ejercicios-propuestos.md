# Ejercicios Propuestos: Introducción a Java

## Presentación

Estos ejercicios permiten practicar la estructura básica de Java, entrada y salida de datos, variables y operaciones aritméticas.

---

## Instrucciones generales

Para cada ejercicio entregar:

1. Análisis de entrada, proceso y salida.
2. Código Java.
3. Prueba de escritorio.
4. Explicación breve.
5. Evidencia de ejecución si el docente lo solicita.

---

# Nivel 5. Reto integrador

## Reto. Ficha académica básica

Crear un programa en Java que lea:

- Nombre del estudiante.
- Código estudiantil.
- Programa académico.
- Asignatura.
- Nota del primer corte.
- Nota del segundo corte.
- Nota del tercer corte.

El programa debe calcular la definitiva con la fórmula:

```text
definitiva = corte1 * 0.30 + corte2 * 0.30 + corte3 * 0.40
```

Y mostrar:

```text
Estudiante: Ana Torres
Código: 2026001
Programa: Ingeniería de Sistemas
Asignatura: Algoritmia y Programación
Definitiva: 4.2
```

## Checklist antes de entregar

- [ ] El archivo se llama igual que la clase pública.

- [ ] Se importó `Scanner` si se leen datos.
- [ ] Las variables tienen nombres claros.
- [ ] El programa compila.
- [ ] El programa ejecuta.
- [ ] La salida es clara.
- [ ] Se hizo prueba de escritorio.
- [ ] El estudiante puede explicar el código.
