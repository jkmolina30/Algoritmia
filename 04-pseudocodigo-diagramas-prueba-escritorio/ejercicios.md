# Reto integrador

## Reto. Ficha de liquidación académica

Diseñar una solución completa para un sistema que calcule el pago de matrícula de un estudiante.

Debe leer:

- Nombre del estudiante.
- Programa académico.
- Valor base de matrícula.
- Valor de carné.
- Valor de seguro.
- Valor de derechos complementarios.
- Porcentaje de descuento.

Debe calcular:

```text
subtotal = matricula + carne + seguro + derechos
descuento = subtotal * porcentajeDescuento / 100
total = subtotal - descuento
```

Debe entregar:

1. Análisis.
2. Lenguaje natural.
3. Pseudocódigo.
4. Diagrama de flujo.
5. Prueba de escritorio con datos inventados.
6. Explicación de la solución.

---

## Criterios de revisión

| Criterio | Descripción |
|---|---|
| Análisis | Identifica entrada, proceso y salida |
| Pseudocódigo | Presenta una secuencia clara |
| Diagrama | Representa correctamente el flujo |
| Prueba de escritorio | Verifica valores paso a paso |
| Claridad | Usa nombres adecuados |
| Orden | Presenta el trabajo limpio y organizado |
| Explicación | Justifica la solución |
