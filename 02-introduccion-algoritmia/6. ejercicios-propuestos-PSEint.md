# Ejercicios Propuestos en PSeInt

## Unidad 2. Introducción a la Algoritmia

Este archivo contiene ejercicios propuestos para resolver en **PSeInt**.

El propósito es que el estudiante practique:

- Análisis de problemas.
- Identificación de entradas, procesos y salidas.
- Escritura de algoritmos en pseudocódigo.
- Pruebas de escritorio.
- Uso básico de variables.
- Operaciones aritméticas simples.

---

## Instrucciones generales

Para cada ejercicio se debe entregar:

1. Enunciado.
2. Análisis de entrada, proceso y salida.
3. Algoritmo en lenguaje natural.
4. Pseudocódigo en PSeInt.
5. Prueba de escritorio.
6. Explicación breve de la solución.

---

# Nivel 2. Ejercicios con fórmulas geométricas

## Ejercicio 11. Área de un rectángulo

Leer la base y la altura de un rectángulo. Calcular y mostrar su área.

Fórmula:

```text
area = base * altura
```

---

## Ejercicio 12. Perímetro de un rectángulo

Leer la base y la altura de un rectángulo. Calcular y mostrar su perímetro.

Fórmula:

```text
perimetro = 2 * (base + altura)
```

---

## Ejercicio 13. Área de un triángulo

Leer la base y la altura de un triángulo. Calcular y mostrar su área.

Fórmula:

```text
area = (base * altura) / 2
```

---

## Ejercicio 14. Área de un cuadrado

Fórmula:

```text
area = lado * lado
```

---

## Ejercicio 15. Perímetro de un cuadrado

Leer el lado de un cuadrado. Calcular y mostrar su perímetro.


## Ejercicio 16. Área de un círculo

Leer el radio de un círculo. Calcular y mostrar su área.

Fórmula:

```text
area = 3.1416 * radio * radio
```

## Ejercicio 17. Longitud de una circunferencia

Leer el radio de una circunferencia. Calcular y mostrar su longitud.

## Ejercicio 20. Nota definitiva con porcentajes

Leer tres notas:

- Primer corte.
- Segundo corte.
- Tercer corte.

Calcular la nota definitiva sabiendo que:

```text
Primer corte = 30%
Segundo corte = 30%
Tercer corte = 40%
```

Fórmula:

```text
definitiva = corte1 * 0.30 + corte2 * 0.30 + corte3 * 0.40
```

## Ejercicio 28. Gasto semanal en transporte

# Reto integrador en PSeInt

## Reto. Calculadora académica básica

Diseñar un algoritmo en PSeInt que lea:

- Nombre del estudiante.
- Nombre de la asignatura.
- Nota del primer corte.
- Nota del segundo corte.
- Nota del tercer corte.

Debe calcular la nota definitiva usando:

```text
Primer corte = 30%
Segundo corte = 30%
Tercer corte = 40%
```

Debe mostrar un mensaje como:

```text
El estudiante Ana obtuvo una definitiva de 4.1 en Algoritmia.
```

## Criterios de revisión

| Criterio | Descripción |
|---|---|
| Análisis | Identifica correctamente entrada, proceso y salida |
| Pseudocódigo | Usa estructura clara en PSeInt |
| Variables | Declara variables con nombres adecuados |
| Operaciones | Aplica correctamente las fórmulas |
| Prueba de escritorio | Verifica la solución con datos de ejemplo |
| Explicación | Describe la solución con sus propias palabras |
