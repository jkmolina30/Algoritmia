# Reto integrador

## Sistema de notas con matriz

Crear un programa en Java que permita registrar notas de varios estudiantes.

El programa debe:

1. Preguntar cuántos estudiantes se van a registrar.
2. Preguntar cuántas notas tiene cada estudiante.
3. Crear una matriz `double[][] notas`.
4. Leer y validar cada nota entre 0.0 y 5.0.
5. Calcular el promedio de cada estudiante.
6. Calcular el promedio general del grupo.
7. Mostrar el estudiante con mayor promedio.
